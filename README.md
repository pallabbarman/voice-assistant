# Voice Assistant

Using your voice command you can do

-   [x] Search anythings in YouTube
-   [x] Search anythings in Wikipedia
-   [x] Search anythings in Google
-   [x] Play Music
-   [x] Open Google
-   [x] Open YouTube
-   [x] Open Apps
-   [x] Check Current Time
-   [x] Listen Jokes

## Installation

For run this project on you computer run the below command:

#### Speech Recognition

`pip install speechRecognition`

#### Text to speech

`pip install pyttsx3`

#### To get Wikipedia info

`pip install wikipedia`

#### To get funny jokes

`pip install pyjokes`

#### Control on browser

`pip install pywhatkit`

#### Python Audio

`pip install PyAudio`

#### If any error occurs

`pip install pipwin`

`pipwin install pyaudio`
